# Futuristic Links Dashboard

This is a fully JavaScript-powered, futuristic links dashboard designed for developers' portfolios. It features:

- AI-style animated background with floating points and connecting lines.
- Fully interactive link cards with hover and click animations.
- Dark/Light theme toggle.
- Responsive layout with grid display (3 columns on desktop, 2 on tablets, 1 on mobile).
- Easy to customize via a single `profileData` JS object.

## Tech Stack

- HTML5 / CSS3
- JavaScript (Vanilla + GSAP)
- Font Awesome Icons
- Responsive Design

## How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Enjoy the futuristic animations and interactive links
