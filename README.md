##  Project Overview

This is a **fully JavaScript-powered futuristic links dashboard** designed to showcase a developer's portfolio or social links in an interactive and visually appealing way.  

**Features:**

- AI-inspired animated background with floating points and connecting lines.  
- Fully interactive link cards with hover and click animations.  
- Dark / Light theme toggle.  
- Responsive grid layout:  
  - Desktop: 3 columns  
  - Tablet: 2 columns  
  - Mobile: 1 column  
- Easy customization through a single `DATA` JavaScript object.  
- Fully self-contained, no backend required.

##  Features in Detail

1. **AI Background Animation**  
   - Canvas-based particle system with connecting lines.  
   - Fully responsive and interactive.  

2. **Interactive Link Cards**  
   - Hover effects: lift, scale and glow.  
   - Click animations with subtle scale feedback.  
   - Customizable icon color, size and text via `DATA.links`.

3. **Profile Section**  
   - Name, title, and tagline displayed prominently.  
   - Neon-glow effect for a futuristic feel.  

4. **Dark / Light Theme Toggle**  
   - Clickable icon toggle in the top-right corner.  
   - Smooth transition between themes.  

5. **Responsive Layout**  
   - Grid adapts to desktop, tablet and mobile screens.

### Customization

##### All links, profile info, and icon colors are controlled via the DATA object in main.js:
```
const DATA = {
  theme: "dark",
  profile: { 
    name: "Muhammad Yasir", 
    title: "Full-Stack Web Developer", 
    tagline: "Building modern, animated & interactive web experiences" 
  },
  links: [
    { name:"LinkedIn", url:"https://linkedin.com/in/yasirawan4831", icon:"fa-brands fa-linkedin-in", color:"#0A66C2" },
    { name:"GitHub", url:"https://github.com/YasirAwan4831", icon:"fa-brands fa-github", color:"#ffffff" },
    ...
  ]
}
```
Simply add/remove links or edit profile info to update the dashboard.
